Things on which we Work.

Algorithms

We work on Algorithms for performing calculations, data processing, automated reasoning, and other tasks. We also optimize our algorithms such that they can be used by millions of people.

Computer Architectures

We study and work on the fundamentals of computer architecture, processors, Chips. We make products using, which common people can learn about the fundamentals of Computers.

ML

We work on many latest ML algorithms and technologies. Use ML applications in our products to help mankind and for easy use.

Game Developmet

We work on many data structures to build new games of new and retro generations. We also build game consoles that contain many games and release them to the market.

Open Source

At GCC we understand the importance of open source contributions. Hence we also determined to contribute to the open-source and also develop open-source products.

Software

At GCC we develop software with the immense power of computation. And help everyone's in their daily lifestyle.


On going Projects.

Pencil

Pencil is a Knowledge Computational Engine, Which give answer to the user's query. A simple, easy to use tool that do search for you, model and calculate for you and show the result for you.

BitKit

BitKit is a kit for building a 16-bit computer. It is very helpful to understand the system architecture. Also Perform many basics tasks e,g: File Management, Arthematic Operations, Games, etc.

Sunday

Sunday (a little teachable object) is an open-source AI experiment. It introduces the basics of machine learning by helping you build your own teachable object.
